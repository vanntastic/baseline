Baseline : A simple, sensible collection of styles

Simply a collection of css files that I use as a baseline for getting started with a project, this is a collection of various files and frameworks such as 960.gs and useful objects to use in a project like icons and such. I have this setup because I didn’t want to have to repeat myself every single time I start working on a project.

What baseline does do:

  • Gives you a grid system based on the 960.gs
  • Leaves the typography up to you
  • Gives you some [optional] awesome button widgets to use

What it doesn't do:

  • Assume it knows what font you like
  • Reset your css for you, well it kinda does, but only on the viewport
